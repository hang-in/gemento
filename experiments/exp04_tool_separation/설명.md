# 실험 4: 도구 호출과 루프 분리

## 목적

도구 호출을 같은 루프에서 처리(단일)할지, 별도 루프로 분리할지 결정한다.

소형 모델(4.5B)이 한 루프에서 "도구 요청 → 결과 해석 → 추론"을 모두 감당할 수 있는지 확인.

## 사전 조건

- 실험 2 (multiloop) 완료 → H1 채택됨
- 실험 2의 config (soft cap 등) 반영됨
- Ollama에 `gemma4:e4b` 설치됨

**주의:** 이 실험은 현재 `run_experiment.py`에서 구조만 잡혀있고 구현이 불완전하다.
실험 2 결과를 본 뒤 도구 시뮬레이션 로직을 완성해야 한다.

## 측정 항목

- 단일 루프 vs 분리 루프의 도구 결과 활용 정확도
- 추론 품질 차이
- 총 소요 루프 수

## 판정 기준

```
품질 차이 < 5%  → 단일 루프 채택 (효율 우선)
품질 차이 ≥ 5%  → 분리 루프 채택 (품질 우선)
```
